# VeriCause – Donation Transparency & Impact Platform

## Overview
VeriCause is a modern frontend web application designed for transparent giving and tracking real-world impact. Built purely for local and offline development, it contains no telemetry, external framework branding, or required cloud services out of the box.

## Technologies Used
- Vite
- TypeScript
- React 18
- shadcn/ui
- Tailwind CSS

## Offline Setup & Execution

### Prerequisites
Node.js (v18+ recommended) & npm

### Getting Started Locally

Install the dependencies (they are cached in this environment):
```sh
npm install
```

Start the development server:
```sh
npm run dev
```

Build the self-contained production bundle:
```sh
npm run build
```

This application functions with complete independence from external services.
